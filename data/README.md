# Murasaki Benchmark 数据集说明

## 数据格式

所有数据文件采用 JSONL (JSON Lines) 格式，每行一条JSON记录。

### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `src` | string | 日语源文本 |
| `ref` | string | 中文参考译文 |
| `category` | string | 类别：`Short` (短文) 或 `Long` (长文) |

### 示例

```json
{"src": "「そうだな、玄霧のほうが顔の造形に隙がない」", "ref": "「是啊，玄雾的脸型比较没有瑕疵。」", "category": "Short"}
```

## 数据集文件

| 文件 | 样本数 | 源文本长度 | 用途 |
|------|--------|------------|------|
| `dataset_short.jsonl` | 100 | 180-200字符 | 短文本翻译评测 |
| `dataset_long.jsonl` | 100 | 780-800字符 | 长文本翻译评测 |

## 数据来源

语料来自日语轻小说及其中文翻译，覆盖以下题材：

- 异世界/转生 (例: 转生史莱姆、无职转生)
- 恋爱喜剧 (例: 俺妹、果然我的青春恋爱喜剧搞错了)
- 科幻/机战 (例: 86不存在的战区、全金属狂潮)
- 奇幻/冒险 (例: 刀剑神域、狼与香辛料)
- 日常/治愈 (例: 邻家天使、义妹生活)

数据来源于100本不同的日语轻小说，确保评测样本的多样性。

## 采样策略

评测数据集的构建采用以下策略：

1. **对齐质量筛选**: 仅选取高质量对齐段落
2. **长度控制**: Short类180-200字符，Long类780-800字符
3. **多样性保证**: 每本书最多贡献1个样本

## 使用建议

直接使用 `dataset_short.jsonl` 和 `dataset_long.jsonl` 进行评测即可。
